# Ejecutar en modo desarrollo
1. Clonar el repositorio
2. Ejecutar el comando:
```
npm i
```
3. Tener Nest CLI instalado
```
npm i -g @nest/cli
```
4. Levantar la base de datos
```
docker-compose -f docker-compose.yml up -d --build --force-recreate
```

## Stack usado:
* MongoDB
* Nest
* Docker
* PostgreSQL
